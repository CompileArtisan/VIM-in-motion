#+title: VIM in Motion


* Modes
- It's a Modal text editor, where each mode changes what your keys do. 
- Your keys don't just type, they also serve as different ways of interacting
  with text through keybinds.
- In VIM, there are 3 modes.
** Normal Mode (default)
- This is where you navigate and edit
- Keys do actions, not typing
- Example:
  - ~w~ is for moving forward a word
  - ~d~ is for deleting (but needs a motion)

** Insert Mode
- This is where you actually type text
  - ~i~ is for inserting before cursor
  - ~a~ is for inserting after cursor
Press ~Esc~ to go back to Normal mode

** Visual Mode
- Used to select text
  - ~v~ is for character selection
  - ~V~ is for line selection

* How a VIM motion works
** The universal template
#+begin_src txt
action + count + motion
#+end_src

*** Actions
- ~d~ for delete
- ~y~ for yank (copy)
- ~v~ is for visual (select)
- ~V~ is for visual line (select but line by line)

*** Motion
- ~w~ for word forward
- ~b~ for word backward
- ~e~ for end of word
- ~$~ for end of line
- ~0~ for beginning of line


*** Combining them
- ~d3w~ means *delete 3 words forward*
- ~dw~ means *delete 1 word forward* (the same as ~d1w~ but in a motion, the count is
  optional)
- ~d$~ means delete till the end of the line
  
** Template with text objects
#+begin_src txt
action + [i/a] + object
#+end_src
*** What is ~[i/a]~
~i~ means inside and ~a~ means around.

*** Object
- ~w~ means word
- ~(~ for paranthesis and ~"~ for double quotes
- ~t~ means tag (XML or HTML)

*** Combining them
- ~vit~ means *select contents inside tag*.
